🚀 API Principal

El endpoint principal para generar reportes personalizados:

POST /api/reports/page

Envía tu configuración de layout (vertical/horizontal), información del header e componentes para generar reportes dinámicos.

📖 Ver Documentación API 🌟 Ejemplo JSON ❤️ Health Check

💡 Uso Rápido

Para generar un reporte personalizado, envía un POST a /api/reports/page con esta estructura:

{
  "layout": "vertical",
  "headerInfo": {
    "institucion": "Tu Institución",
    "evaluacion": "SABER 11° - 2025",
    "municipio": "Tu Ciudad",
    "fecha": "31/07/2025"
  },
  "chartTitle": "Título de tus gráficos",
  "components": [
    {
      "type": "bar_chart_with_title",
      "data": { /* datos del gráfico */ }
    }
  ]
}
